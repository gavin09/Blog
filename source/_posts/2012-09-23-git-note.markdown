在這邊記錄平常會使用到的git 指令。

##安裝git##

	sudo apt-get install git-core

##產生新的branch##

	git branch [branch_name]

##git 切換##

	git checkout [branch_name]

##git tree##

	git log --graph --pretty=oneline --abbrev-commit
