Using Latex With Command Line

Feb 23rd, 2013 | Comments

Installation

$ sudo apt-get install texlive-full

command

$ latex name.tex
$ bibtex name (generates name.bbl)
$ latex name.tex
$ pslatex name.tex
$ dvipdf name.dvi
