Dropbox-auto-run

mkdir $HOME/.config/autostart

vi Dropbox.desktop

[Desktop Entry]
Encoding=UTF-8
Name=Dropbox
Comment=RunDropbox
Icon=/usr/share/icons/elementary/panel/22/dropboxstatus-logo.svg
Exec=bash /home/*user*/.dropbox-dist/dropboxd
Terminal=false
Type=Application

Note:

Icon need to be checked
